# 财务自由计算器

🔗 **在线体验**: [fire-zeta.vercel.app](https://fire-zeta.vercel.app)

给定你的当前年龄、当前存款、预期寿命、年化通胀率、年化名义理财回报率，计算在剩余寿命内可持续的、以今日购买力计的每月可支配收入。

## 核心公式

- 真实年化收益率 \( r = \frac{1+R}{1+\pi} - 1 \)，其中 \( R \) 为名义年化回报率，\( \pi \) 为年化通胀率
- 月度真实收益率 \( r_m = (1+r)^{1/12} - 1 \)
- 月数 \( N = 12 \times (\text{预期寿命}-\text{当前年龄}) \)
- 每月可支配收入（今日购买力）：\( w = PV \cdot \dfrac{r_m}{1 - (1+r_m)^{-N}} \)

## 功能特性

- ✅ 计算每月可支配收入（今日购买力）
- ✅ 反推目标退休所需存款
- ✅ 资产余额与月收入可视化图表
- ✅ 实时美元汇率转换
- ✅ 暗色/亮色主题切换
- ✅ URL 参数分享
- ✅ localStorage 数据持久化
- ✅ 图片导出（含参数水印）
- ✅ 社交平台分享（微信/小红书等）

## 本地开发

```bash
# 启动本地服务器
python3 -m http.server 8000

# 访问
open http://localhost:8000/web/index.html
```

## 文件结构

```
web/
  index.html   # 页面结构
  styles.css   # 样式（含暗色主题）
  config.js    # 配置：颜色、字号、对数轴范围
  calc.js      # 计算逻辑：利率换算、收入/资产序列
  chart.js     # Chart.js 双轴图表渲染
  app.js       # 交互逻辑：表单、计算、分享、导出
```

## 说明

- 输出金额为"今日购买力"，便于与当前物价对照
- 模型假设领取到预期寿命末尾，终值为 0
- 本工具不构成投资建议
